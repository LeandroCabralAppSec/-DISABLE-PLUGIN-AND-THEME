/** # DESABILITAR INSTALAÇÃO DE PLUGINS E TEMAS # */

Uma dica rápida que ajuda a restringir o acesso aos plug-ins e do tema a atualizar o recurso no WordPress, além de remover a capacidade de os 
usuários instalarem novos plug-ins e temas.

/ ** # DISABLE PLUGIN AND THEME INSTALLATION # * /

A quick tip that helps restrict access to plug-ins and the theme to update the feature on WordPress, in addition to removing the ability for 
users to install new plug-ins and themes.
